# **App Name**: Ward Dashboard

## Core Features:

- User Authentication: Secure login and authentication using Firebase Auth.
- Role-Based Access Control: Control user access based on roles (bishop, counselor, secretary) using Firestore and Firebase Security Rules.
- Interview Management: Create, read, update, and delete interview schedules and records.
- Bishopric Meeting Notes: Record and manage notes from bishopric meetings.
- Sacrament Meeting Agenda: Create, manage, and export sacrament meeting agendas to PDF.
- Activity logger: Log all user actions (create, update, delete, login) and store them in Firestore.
- Reasoning tool: Tool that allows AI to suggest improvements to sacrament agendas based on ward needs and past meeting data, optimizing themes and speakers.

## Style Guidelines:

- Primary color: Institutional blue (#20639B) to evoke trust and professionalism.
- Background color: Light gray (#F0F4F8), a desaturated tint of the primary blue, for a clean and unobtrusive backdrop.
- Accent color: Muted gold (#A38255), analogous to institutional blue, to highlight key interactive elements.
- Body and headline font: 'Inter' for a modern, neutral, professional look. Inter is sans-serif and appropriate for both headers and body copy.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use 'lucide-react' icons for a clean and consistent design.
- Dashboard layout with left sidebar, top header, and main content area for a professional and organized look.
- Subtle animations using Framer Motion to enhance user experience without being distracting.
